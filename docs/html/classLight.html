Detailed Description

[Light] class of the light (inherits from RaySetting) This class is used to define the light and to define the methods that the light must have (ex: addition, subtraction, ...)

Parameters
_ambientthe ambient light (0.0 - 1.0)
_diffusethe diffuse light (0.0 - 1.0)
_pointLightsthe point light sources (Vector3D)
_directionalLightsthe directional light sources (Vector3D)
