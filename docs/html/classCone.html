Detailed Description

[Cone] class of the cone primitive (inherits from Iprimitives) This class is used to define the cone primitive and to define the methods that the cone must have (ex: intersect, ...)

Parameters
_positionthe position of the cone (Vector3D)
_directionthe direction of the cone (Vector3D)
_radiusthe radius of the cone (float)
_heightthe height of the cone (float)
_colorthe color of the cone (Color)
